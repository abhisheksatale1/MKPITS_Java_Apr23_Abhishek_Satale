What is Bootstrap?

  • Boostrap is an open source frontend framework.
  • It's totally free for everything
  • Bootstrap includes HTML and CSS based design templates for typography, forms, buttons, tables, navigation, modals, image carousels and many other, as well as optional JavaScript plugins and JQuery widgets
  • Bootstrap also gives you the ability to easily create responsive designs which is dispay on any electronic or display devices. Like mobile, tab, laptop, PC, etc

What is responsive website?

  • Responsive web design (RWD) or responsive design is an approach to web design that aims to make web pages render well on a variety of devices and window or screen sizes from minimum to maximum display size to ensure usability and satisfaction
  • Responsive web design became more important as users of mobile devices came to account for the majority of web site visitors.[13][14] In 2015, for instance, Google announced Mobilegeddon and started to boost the page ranking of mobile-friendly sites when searching from a mobile device.
